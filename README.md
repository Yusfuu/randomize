## Installation

Use the package manager [yarn](https://yarnpkg.com/) to install.

```bash
yarn
```

## Dev

For Starting development server

```bash
yarn dev
```

## Build

For Optimized Build 

```bash
yarn build
```

## Test

For Testing

```bash
yarn test
```
